~~Video-Game Database~~

Classes:
	- BST - data structure used to manage data
	- Hash Table - data structure used to manage data
	- List - Required for hash table operations (Collisons)
	- Game - Contains fields and methods to operate the database
	- GameDatabase(Main) - Puts all the classes together with a UI to make database
How To Run:

Create a new project and put all 5 classes in the same package in the project. GameDatabase is the main class so run that.
